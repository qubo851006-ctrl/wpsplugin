====================================================
  WPS AI智能问答助手  安装与使用说明
====================================================

【前置要求】
  - WPS Office（已安装）
  - Python 3.x（已安装，且已安装 httpx、python-dotenv）
  - D:\claude\training-manager\.env 中存有 API 配置


【第一步：启动代理服务器】

  方式 A（推荐）：双击 "启动.bat"
  方式 B：在项目目录打开命令行，运行：
            python proxy.py

  启动后终端会显示：
    WPS AI助手 代理服务器
    地址: http://127.0.0.1:8765
  代理服务器需要保持运行，不要关闭窗口。


【第二步：在 WPS 中安装加载项】

  方式一（推荐 - WPS 开发者工具）
  ─────────────────────────────────
  1. 打开任意 WPS 文档
  2. 顶部菜单 → "开发工具" → "加载项管理器"
     （若没有开发工具：文件 → 选项 → 自定义功能区 → 勾选"开发工具"）
  3. 点击 "添加" 或 "从文件安装"
  4. 选择本目录下的 manifest.xml 文件
  5. 确认安装

  方式二（wpsjs 命令行）
  ─────────────────────────────────
  1. 安装 wpsjs：npm install -g wpsjs
  2. 在本目录打开命令行，运行：wpsjs debug
  3. wpsjs 会自动将加载项注册到 WPS 并打开

  方式三（手动复制配置文件）
  ─────────────────────────────────
  将 manifest.xml 复制到 WPS 加载项目录：
    C:\Users\{用户名}\AppData\Roaming\kingsoft\wps\jsaddons\
  然后重启 WPS。


【第三步：使用 AI 助手】

  安装成功后，WPS 功能区会出现 "AI 助手" 标签页。

  1. 打开 AI 助手
     点击 "AI 助手" → "AI 助手" 大按钮
     → 右侧弹出 AI 聊天侧边栏

  2. 普通对话
     直接在输入框输入问题，Ctrl+Enter 发送
     AI 使用内网 glm-5-outside 模型回答

  3. 引用文档内容
     - 在文档/表格/幻灯片中选中文字
     - 点击 "引用选中" 按钮（侧边栏内）
       或点击功能区 "引用选中" 按钮
     → 选中内容会附加到下一条发送的消息中

  4. 知识库模式
     点击侧边栏右上角 📚 按钮切换
     → 切换后查询走智弈智枢知识库


【支持的 WPS 应用】

  WPS Writer（文字）   ✓
  WPS ET（表格）       ✓
  WPS Presentation（演示） ✓


【故障排查】

  问题：侧边栏显示 "无法连接到代理服务器"
  解决：检查 proxy.py 是否正在运行（看命令行窗口）

  问题：AI 回复报错 500
  解决：检查内网 VPN 是否连接；检查 API Key 是否有效

  问题：知识库无响应
  解决：确认内网可访问 10.151.62.18

  问题：WPS 中看不到 "AI 助手" 标签页
  解决：重新安装加载项；或重启 WPS
